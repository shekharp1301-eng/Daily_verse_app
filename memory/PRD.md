# PRD — Daily Verse: Today’s God’s Word

## Problem Statement
Build a clean, modern, spiritual mobile app that delivers a daily God’s Word experience with bilingual support (English + Telugu), AI-generated daily devotional content, user accounts, favorites/history, streak tracking, and push notifications.

## Architecture
- **Frontend:** Expo SDK 54 + React Native (expo-router single-route shell, componentized screens)
- **Backend:** FastAPI + Motor (MongoDB)
- **Database:** MongoDB (`users`, `verses`, `favorites`, `read_logs`)
- **Auth:** JWT bearer tokens with bcrypt password hashing
- **AI Content:** `emergentintegrations` LLM chat (OpenAI gpt-5.2 via `EMERGENT_LLM_KEY`), with safe fallback content
- **Notifications:** Expo push token registration + send-test/send-daily endpoints + background dispatch loop

## User Personas
1. **Daily Devotional Reader** — wants one peaceful verse experience every morning.
2. **Bilingual Believer** — wants to read in English or Telugu instantly.
3. **Habit Builder** — wants streak visibility and reminders to stay consistent.

## Core Requirements (Static)
- Splash, Home, History, Favorites, Settings screens
- AI-generated daily verse content (verse, reference, theme, explanation, message, prayer)
- EN/Telugu language support
- User signup/login with cloud sync
- Favorite/save and verse history
- Daily streak tracking
- Notification preferences + push delivery support
- Modern spiritual visual style with image-backed verse card
